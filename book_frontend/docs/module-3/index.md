## Module Structure

**6 Chapters** on advanced simulation, perception, and AI training:

### Chapter 3.1: Introduction to Isaac Sim
Install Isaac Sim, explore USD (Universal Scene Description), and spawn your first robot.

### Chapter 3.2: Building Photorealistic Environments
Create detailed worlds with USD, import CAD models, and apply PBR materials.

### Chapter 3.3: Advanced Sensor Simulation
Simulate RGB-D cameras, semantic segmentation, and GPU-accelerated LiDAR.

### Chapter 3.4: Synthetic Data for Computer Vision
Generate training datasets with automatic labeling for object detection and segmentation.

### Chapter 3.5: Reinforcement Learning Integration
Train RL agents with Isaac Gym, leverage GPU parallelization for policy learning.

### Chapter 3.6: Deployment to Real Robots
Bridge simulation-to-reality gap with domain randomization and sim-to-real transfer.
